# <a name="azure-mobile-libraries-for-net"></a>用于 .NET 的 Azure 移动库

## <a name="client-library"></a>客户端库

直接从 Visual Studio [包管理器控制台][PackageManager]或使用 [.NET Core CLI][DotNetCLI] 安装 [NuGet 包](https://www.nuget.org/packages/Microsoft.Azure.Mobile.Server)。

#### <a name="visual-studio-package-manager"></a>Visual Studio 包管理器

```powershell
Install-Package Microsoft.Azure.Mobile.Server
```

```bash
dotnet add package Microsoft.Azure.Mobile.Server
```
